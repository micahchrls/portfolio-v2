# Micah Mustaham's Portfolio

A modern, responsive portfolio website built with React, TypeScript, and Vite showcasing my experience as a Backend Software Developer.

## 🚀 Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
- **Dark/Light Mode** - Theme switching with persistent user preference
- **AI Chatbot** - Interactive chatbot powered by Google Gemini AI
- **GitHub Integration** - Real-time GitHub metrics and statistics
- **Smooth Animations** - Framer Motion animations for engaging UX
- **Modern UI** - Built with Tailwind CSS and shadcn/ui components

## 🛠️ Tech Stack

- **Frontend:** React 18, TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui, Radix UI
- **Animations:** Framer Motion
- **Icons:** Lucide React, Tabler Icons
- **AI Integration:** Google Generative AI (Gemini)
- **Analytics:** Vercel Analytics

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/micahchrls/portfolio-v2.git

# Navigate to project directory
cd portfolio-v2

# Install dependencies
npm install

# Start development server
npm run dev
```

## 🔧 Environment Variables

Create a `.env` file in the root directory:

```env
VITE_GEMINI_API_KEY=your_google_gemini_api_key
VITE_GITHUB_USERNAME=micahchrls
```

## 📝 Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
